    给函数 Bind 对象

    2018-08-06

    我们常常需要将一个对象绑定到一个方法的 this 上。在 JS 中，如果你想要调用一个函数并指定它的 this 时可以使用 bind 方法。

    Bind 语法

    #参数

    thisArg

    当绑定函数被调用时，该参数会作为原函数运行时的 this 指向。

    arg1, arg2, …

    当绑定函数被调用时，这些参数将置于实参之前传递给被绑定的方法。

    返回值

    返回由指定的this值和初始化参数改造的原函数拷贝
